MyFlix

The objective of this api is to focus on the server side componenets of a movies web application. The web application will provide information about movies, directors and different genres to users. Users will have the option to sign up with their personal information and create a list of their favorite movies.

Request Url Http Method Request Body Response Body
Get list of Movies /movies GET None Json objects show data on list of movies
Movie details /movies/[movietitle] GET None Json objects showing data on movie title, img, description,genre, directors, actors
Movie Genres /movies/[genre] Get None Json objects returns movies based on genres
Movie Directors /movies/[directors] Get None Json objects returns information on directors, birth, death
Favorite Movies Get /user/[username]/movies/[favorite]
Add to favorites /user/[username]/movies/[[favorite]/[moviesId] Post Json object about movie being added Json object with the movie that was added and an id
Delete movies from favorites /user/[username]/movies/[favorite]/remove/[moviesId] Delete None message indicating the status on if the movie was deleted or not
Add a New User /user Post Json about new username Json with information added and id confirming completion or failure
Log in /user/[username] Get None Json with data about users account
Update account Post /user/[username] Json with data that is being added Json confirming that updates have been applied or denied
Delete a user /user/[username] Delete None Message indicating the status of whether or not the request was completed
